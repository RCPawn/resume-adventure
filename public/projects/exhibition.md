# 🏛️ 非遗虚拟展陈与数据监测平台

> 文化遗产数字化转型的 Web 3D 可视化探索

---

## 📖 项目概述

在文化遗产保护领域，如何将珍贵的非物质文化遗产进行数字化呈现，是技术人面临的挑战。本项目基于 Vue 3 + Spring Boot 技术栈，融合 **3D 可视化、交互式展陈、数据驾驶舱** 三大核心能力，打造白族非遗文化的数字孪生平台。

### 技术挑战与突破

- 🎨 **3D 沉浸体验**：基于 Google Model Viewer 实现展品 360° 交互展示
- 📜 **创新交互设计**：首创 3D 翻书式传承图谱，图谱化呈现师徒传承脉络
- 📊 **数据驱动决策**：ECharts 多维可视化，实时监测文化传播热度
- 🌐 **UGC 内容生态**：用户生成内容体系，促进全民参与文化保护

---

## 🏗️ 系统架构

### 整体架构

系统采用前后端分离架构，前端基于 Vue 3 + Vite 构建，后端采用 Spring Boot 微服务架构。

**核心模块**：
- 🌏 **数字驾驶舱**：数据总览中枢，ECharts 多维可视化 + 3D 场景融合
- 🏛️ **在线展厅**：3D 沉浸展陈，支持旋转缩放、点赞收藏互动
- 📜 **传承图谱**：3D 翻书交互设计，力导向图展示传承关系网络
- 📸 **纸上乾坤**：瀑布流影像档案馆，沉浸式看图模式
- 🎵 **三道余音**：音频播放与波形可视化，视听融合体验

```
📦 非遗虚拟展陈平台
├── 🌏 数字驾驶舱          [数据总览中枢]
├── 🏛️ 在线展厅            [3D 沉浸展陈]
├── 📜 传承图谱            [3D 翻书交互] 
├── 📸 纸上乾坤            [影像档案馆]
├──  三道余音            [声音档案]
└── ️ 后台管理系统        [内容管理]
```

## 🔍 核心功能实现

### 🌏 数字驾驶舱 - 数据总览中枢

**设计目标**："一屏观全域，一网管全局"

数字驾驶舱作为系统的智慧大脑，通过 **ECharts 数据可视化 + 3D 场景融合** 技术，实时呈现非遗资源的分布态势与传播效果。

**核心指标体系**：
- 📊 非遗项目总量与分类占比
- 🔥 累计浏览热度与互动趋势
- 🏆 Top 5 热门藏品排行榜
- ☁️ 非遗热词云图
- 🎯 最新收录展品动态

### 🎪 在线展厅 - 3D 沉浸展陈

**技术创新**："指尖旋转千年工艺，屏幕触摸历史温度"

基于 **Google Model Viewer** 构建的 3D 展品展示系统，实现：
- 🔄 360° 自由旋转与缩放
- 💡 真实光影渲染
- 📱 移动端适配
- ❤️ 点赞收藏与评论互动

**展品详情页优化**：
- 紧凑式统计数据展示（浏览量、点赞、收藏）
- SVG 矢量图标（爱心❤️ / 星星⭐），点击填充动画
- 渐变标题与圆角标签设计
- 响应式布局适配多端

- 🌳 师徒传承关系链
- 🏅 国家级/省级/市级传承人分级标识
- 💬 悬浮信息卡片展示详细档案

## 🎨 设计亮点

### 1. 黑白极简美学
遵循 **白族"尚白"文化传统**，界面以黑白灰为主色调，辅以渐变色点缀，营造素雅清新的视觉体验。

### 2. 交互动画优化
- ❤️ 点赞心跳动画
- ⭐ 收藏闪烁效果
- 📖 3D 翻书过渡
- 🖼️ 图片懒加载与平滑滚动

### 3. 响应式设计
完美适配桌面端、平板、手机等多尺寸设备，确保最佳浏览体验。

### 4. 性能优化
- 3D 模型 GLB 格式压缩
- 图片懒加载与 CDN 加速
- 数据库索引优化
- Redis 缓存热点数据
